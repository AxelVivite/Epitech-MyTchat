components/atoms:
- Encapsulation d'un élément
Exemple: <input> -> <Input>

components/molecules:
- Aglomération d'atomes et d'éléments
Exemple: <Input> + <ButtonIcon> -> <SearchBar>

components/views:
- Page / Screens / Views
Exemple: <Home>

components/layout:
- Pattern réutilisable
Exemple: <Header>
